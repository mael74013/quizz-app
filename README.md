                "# Quizz App" 

## DESCRIPTION
Quizz App est une application de quiz intéractive ou nous avons mis en avant l'utilisation des langages tels que :
* Le HTML
* Le CSS
* Le java script
Notre application à été développer dans github pour nous permettre un travail plus facile et egalement un assemblage de nos idées et compétences chacune. Le travail comporte sur github 7 branches et plus de 30 commits.

## Repartition du travail
Les taches ont été diviser comme suites:
* Mfossa c'est charger de la création de l'interface et de la stylisation (HTML & CSS) en créant les branche (main, develop, feature/quizz-ui) ou a l'intérieur de chacune des branche est reparti des commits et codes
* DongDing c'est charger de la génération des questions , reponses, timer(Java script) en créant les branches (JS, Js.core, html, style.css)

## Organisation du projet
 Notre projet est organiser comme suite : 
 * La branche main contient la version final du projet avec tous les codes corrects et verifier
 * La branche develop contient l'une des version de CSS proposer
 * La branche feature/quizz-ui est la branche où à été developper l'interface HTML
 * la branche JS et la branche Js.core pour le developpement du java script
